# auraflow-digital

Vite-based React application.

The repository was previously using a placeholder README. This version keeps the documentation short until the implementation is documented in the codebase.

## Local development

```bash
npm install
npm run dev
npm run build
```

## Architecture

This is a client-side React application built by Vite. Components and local assets live under `src`, and the production build emits static files; the repository does not include a backend or persistent data layer.
